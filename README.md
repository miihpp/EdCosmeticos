# EdCosmeticos
Esse Website foi feito para a Ed Cosméticos.

A **Ed Cosméticos** é uma loja especializada em produtos de beleza e cuidados pessoais, oferecendo uma variedade de itens para cabelo, rosto, acessórios, perfumes e muito mais. Este projeto foi desenvolvido com o objetivo de criar uma presença digital moderna para a empresa, facilitando a divulgação dos produtos e proporcionando uma experiência prática para os clientes.
