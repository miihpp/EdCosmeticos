# EdCosmeticos
Esse Website foi feito para a Ed Cosméticos.

A **Ed Cosméticos** é uma loja, localizada em Severínia, São Paulo, especializada em produtos de beleza e cuidados pessoais, oferecendo uma variedade de itens para cabelo, rosto, perfumes (importados e nacionais), acessórios e muito mais. Este projeto foi desenvolvido com o objetivo de criar um site digital para a empresa, facilitando a divulgação dos produtos e proporcionando uma experiência prática para os clientes, que preferem comprar direto de sua casa.
